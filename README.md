# Triumph Kia Teh Portfolio

## What This Is

A Next.js App Router site that statically exports to GitHub Pages.

## Project layout

- `app/`, `components/`, `lib/`: Next.js source
- `public/images/`: images used by the site
- `docs/`: architecture notes and reference docs
- `legacy/static/`: the older plain HTML/CSS/JS site kept for reference
- `out/`: static export output (generated by build)

## Local development

```bash
npm install
npm run dev
```

## Build and static export (GitHub Pages)

```bash
npm run build
```

This produces a static site in `out/`.

## GitHub Pages deployment

Deployment is automated via `.github/workflows/deploy.yml` on pushes to `main`.

## Formspree (contact form)

1. Go to https://formspree.io and create a free account
2. Create a new form, copy the form ID (looks like `xpwzabcd`)
3. In `lib/siteData.ts`, replace `YOUR_FORMSPREE_ID` in `site.contact.form.formspreeAction`
4. Commit and push
